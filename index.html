HTML Basics

HTML (HyperText Markup Language) is the standard language for creating webpages.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My HTML Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
</body>
</html>

HTML Headings

Headings in HTML range from h1 (largest) to h6 (smallest):

<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>
<h3>This is a Heading 3</h3>
<h4>This is a Heading 4</h4>
<h5>This is a Heading 5</h5>
<h6>This is a Heading 6</h6>

HTML Paragraphs

Paragraphs are defined using the <p> tag:

<p>This is a simple paragraph.</p>
<p>Another paragraph goes here.</p>

HTML Lists

You can create ordered and unordered lists using <ul> (unordered) and <ol> (ordered) tags:

<ul>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ul>

<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

HTML Forms

HTML forms allow you to collect user input:

<form action="/submit" method="POST">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    <br>
    <input type="submit" value="Submit">
</form>

HTML Images and Videos

Adding media like images and videos in HTML is simple:

<img src="https://iili.io/dgimjHv.jpg" alt="Sample Image" width="300" height="300">
<video controls width="300" height="300">
    <source src="https://interactive-examples.mdn.mozilla.net/media/cc0-videos/flower.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

HTML Links

Links are created using the <a> tag:

<a href="https://abopu321.github.io/Afrox/">Visit Afrox.com</a>
<a href="https://abopu321.github.io/Afrox/">afrox2.0</a>
<a href="#top">Back to Top</a>

HTML Tables

Tables are used to organize data in rows and columns:

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>25</td>
    </tr>
    <tr>
        <td>Bob</td>
        <td>30</td>
    </tr>
</table>

HTML Attributes

HTML attributes provide additional information about HTML elements:

<a href="https://www.example.com" target="_blank">Visit Example.com</a>
<img src="image.jpg" alt="A beautiful landscape" width="300" height="300">
<input type="text" name="username" id="username" required>

Block vs Inline Elements

HTML elements are typically either block-level or inline:

<!-- Block-level elements -->
<div>This is a block-level element</div>
<p>This is another block-level element</p>

<!-- Inline elements -->
<span>This is an inline element</span>
<a href="#">This is also an inline element</a>
